# 🎨 Gargi Biswas - 3D Creative Portfolio

A stunning, immersive 3D portfolio website that showcases UI/UX design skills with cutting-edge web technologies and artistic vision.

## ✨ Features

### 🚀 **3D Interactive Background**
- **Three.js powered particle system** with 1000+ animated particles
- **Mouse-responsive 3D elements** that react to user movement
- **Smooth particle rotation** and dynamic color gradients
- **Performance optimized** with efficient WebGL rendering

### 🎭 **Artistic Design Elements**
- **Modern dark theme** with purple accent gradients
- **Floating geometric shapes** with smooth animations
- **Glassmorphism effects** with backdrop blur and transparency
- **Custom scrollbar** with accent color theming
- **Interactive hover states** with scale and glow effects

### 🌟 **Advanced Animations**
- **GSAP-powered animations** with ScrollTrigger integration
- **Staggered reveal effects** for content sections
- **Parallax scrolling** for immersive depth
- **Smooth transitions** between sections
- **Loading screen** with animated progress bar

### 📱 **Responsive & Accessible**
- **Mobile-first design** with touch-friendly interactions
- **Smooth mobile navigation** with slide-out menu
- **Keyboard navigation** support for accessibility
- **High contrast mode** support
- **Reduced motion** preferences respected

### 🎯 **UI/UX Showcase**
- **Interactive project cards** with hover overlays
- **Skill visualization** with animated icons
- **Process timeline** with step-by-step breakdown
- **Contact methods** with visual feedback
- **Social media integration** with hover effects

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript ES6+** - Modern JavaScript with modules and async/await
- **Three.js** - 3D graphics and WebGL rendering

- **Intersection Observer API** - Performance-optimized scroll animations

## 🔧 **Setup & Installation**

1. **Clone the repository**
   ```bash
   git clone [your-repo-url]
   cd gargi_portfolio1
   ```

2. **Open in browser**
   - Simply open `index.html` in a modern web browser
   - All dependencies are loaded via CDN

3. **Customize content**
   - Update project images in `assets/` folder
   - Modify text content in `index.html`
   - Adjust colors in CSS custom properties
